Install QuickClaw

The fastest way to get OpenClaw running on your Mac.
Three steps and you're done.

1

Download QuickClaw

Grab the installer from GitHub Releases:

📦 QuickClaw-1.0.0-arm64.dmg ~91 MB

Once downloaded, open the .dmg and drag QuickClaw into your Applications folder.

2

Allow QuickClaw to Open

macOS will show a security warning the first time you open QuickClaw. This is normal and only happens once. Use one of these methods:

Option A — Right-click to Open (easiest)

Find QuickClaw in your Applications folder. Right-click (or Control-click) the app and choose Open. A dialog will appear — click Open to launch it.

Option B — System Settings

Double-click QuickClaw (macOS will block it). Then go to System Settings → Privacy & Security, scroll down, and click Open Anyway next to the QuickClaw message.

Option C — Terminal command (most reliable)

Open Terminal (press Cmd + Space, type "Terminal", hit Enter) and paste this command:

xattr -cr /Applications/QuickClaw.app

Press Enter, then open QuickClaw normally. This is a one-time fix.

💡 Why does this happen?

QuickClaw is signed with an Apple Developer certificate but not yet notarized. macOS quarantines apps that aren't notarized as a security measure. Allowing it once is all you need — after that, it opens like any other app. QuickClaw is fully open-source so you can always verify the code.

3

Launch QuickClaw

Open QuickClaw from your Applications folder (or press Cmd + Space and type "QuickClaw"). The setup wizard will walk you through everything from here.

✓ That's it!

After the first setup, just open QuickClaw like any other app.
No terminal, no extra steps — it just works.
